# dotfiles
my dotfiles and setup scripts

## ArchLinux
### Install
```shell
$ curl 'https://raw.githubusercontent.com/Tosuke/dotfiles/master/bootstrap.sh' | bash -s arch
```
### Requirements
- pacaur
- curl
- bash

## Other *nix os
### Install
```shell
$ curl 'https://raw.githubusercontent.com/Tosuke/dotfiles/master/bootstrap.sh' | bash
```
### Requirements
- curl
- bash
- fish
- ghq
